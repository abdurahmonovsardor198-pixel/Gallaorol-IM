# G‘allaorol IM

Bu repository **G‘allaorol tumani ixtisoslashtirilgan maktabi** rasmiy saytini o‘z ichiga oladi.  

Sayt quyidagi bo‘limlarga ega:  
- **Home** – xush kelibsiz sahifasi va ijtimoiy tarmoqlar linklari  
- **Maktab haqida** – maktabning umumiy ma’lumotlari va yutuqlari  
- **Ustozlar** – o‘qituvchilar ro‘yxati, fanlari va natijalari  

## Foydalanish

1. Repository-ni klonlash yoki ZIP qilib yuklab olish  
2. **index.html**, **style.css**, **script.js** fayllari bilan ishlash  
3. Saytni Netlify yoki boshqa hostingda joylashtirish  

## Texnologiyalar

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome ikonkalari
